# Gambrary!
This is a CRUD Desktop GUI application where it caputures all of my game Information, when I add to my collection and puts it in a excel-sheet, all with just the click of a button! 

## Installation

Use the package manager [pip](https://pypi.org/project/openpyxl/) to install openpyxl.

```bash
pip install openpyxl
